#### **Вопросы формирующего оценивания**

1.  **Данные, информация, знания:** Знание — это осмысленная информация, позволяющая ИИ принимать решения. Цепочка: **Данные** (сырые факты) → **Информация** (структурированные данные) → **Знание** (применение информации для решения задач).
2.  **Пример из жизни:** Магазин собирает **данные** о чеках. Анализ создает **информацию** (отчет о популярных товарах). На ее основе формируется **знание** (стратегия персонализированных предложений для увеличения продаж).
3.  **Классификация:**
    *   **Данные:** "23, 50, 11" — сырые факты.
    *   **Информация:** "Отчет продаж за май: 23 единицы товара А" — структурированные данные.
    *   **Знание:** "Для увеличения продаж товара А запустить скидку" — применение информации для действия.
4.  **Ассоциативное правило:** Правило «хлеб → молоко» ценно, если у него высокая **поддержка** (эта пара часто покупается вместе) и **достоверность** (молоко покупают в 90% случаев с хлебом). Это позволяет использовать перекрестные продажи.
5.  **Тип онтологии:** Онтология **предметной области**. Она содержит детальные медицинские понятия (симптомы, болезни) и их связи, что необходимо для точных диагнозов.
6.  **Компонент ЭС:** **База знаний**. Она содержит все правила и факты, на основе которых механизм вывода строит заключения. Без качественной базы знаний система бесполезна.
7.  **Цепочки рассуждений:**
    *   **Прямая:** От фактов к цели. *Пример: есть симптомы (температура, кашель) → система выводит диагноз (грипп).*
    *   **Обратная:** От гипотезы к фактам. *Пример: есть гипотеза (аппендицит) → система проверяет наличие симптомов (боль, тошнота).*
    *   **Задача для ЭС:** Диагностика неисправности автомобиля. *Приобретение знаний: интервью с автомеханиками, изучение документации.*

2.  **Формулы:**
    *   **Поддержка (A→B):** (Количество транзакций с {A, B}) / (Общее количество транзакций).
    *   **Достоверность (A→B):** (Количество транзакций с {A, B}) / (Количество транзакций с {A}).

3.  **Метод рассуждений:** **Прямая цепочка.** Система от фактов (действия пользователя) идет к заключению (причина неисправности).

4.  **Код (добавление правила):**
    ```python
    # База знаний как список строк-правил
    knowledge_base = []
    def add_rule(rule):
        # Добавляет новое правило в базу знаний
        knowledge_base.append(rule)
        print(f"Правило добавлено: {rule}")

    # Пример использования
    add_rule("ЕСЛИ 'температура > 38' ТО 'возможна инфекция'")
    ```

5.  **Код (поиск факта):**
    ```python
    def find_fact(knowledge_base, pattern):
        # Ищет факт, содержащий pattern, в базе знаний
        for fact in knowledge_base:
            if pattern in fact:
                return fact
        return None

    # Пример использования
    fact = find_fact(knowledge_base, "температура")
    print(f"Найден факт: {fact}")
    ```

    ```python
    from apyori import apriori

    transactions = [
        ['Хлеб', 'Молоко', 'Яйца'], ['Хлеб', 'Сыр', 'Колбаса'],
        ['Молоко', 'Йогурт', 'Бананы'], ['Хлеб', 'Молоко', 'Сыр', 'Помидоры'],

    ]
    # Минимальная поддержка 2 транзакции из 8 -> 2/8 = 0.25
    results = list(apriori(transactions, min_support=0.25, min_confidence=0.5))

    print("Частые наборы и правила (поддержка >= 2):")
    for rule in results:
        items = ', '.join(rule.items)
        support = rule.support
        print(f"- Набор: {{{items}}}, Поддержка: {support:.2f} ({int(support*8)} транзакций)")
        for ordered_stat in rule.ordered_statistics:
             confidence = ordered_stat.confidence
             lift = ordered_stat.lift
             base = ', '.join(ordered_stat.items_base)
             add = ', '.join(ordered_stat.items_add)
             if base: # Выводим только правила, а не наборы
                print(f"  Правило: {{{base}}} -> {{{add}}}, Достоверность: {confidence:.2f}")
    ```

12. **Экспертная система для ПК:**
    *   **а) База знаний (фрагмент):**
        ```python
        knowledge_base = {
            "перегрев": {
                "symptoms": ["высокая температура", "медленная работа", "самопроизвольное выключение"],
                "recommendations": "Проверить работу кулеров, очистить пыль, заменить термопасту."
            },
            "вирус": {
                "symptoms": ["медленная работа", "всплывающие окна", "самопроизвольные перезагрузки"],
                "recommendations": "Запустить полную проверку антивирусом, удалить найденные угрозы."
            },
            "неисправность ОЗУ": {
                "symptoms": ["синий экран смерти", "зависания", "не загружается ОС"],
                "recommendations": "Проверить планки памяти, переставить их в другие слоты или заменить."
            },
            "проблема с жестким диском": {
                "symptoms": ["шум/щелчки из корпуса", "зависания при обращении к файлам", "ошибка загрузки ОС"],
                "recommendations": "Срочно скопировать важные данные и заменить диск."
            }
        }
        ```
    *   **б) Механизм вывода (прямая цепочка):**
        ```python
        def diagnose_pc(user_symptoms):
            possible_diagnoses = []
            for diagnosis, info in knowledge_base.items():
                matching_symptoms = set(user_symptoms) & set(info["symptoms"])
                if matching_symptoms:
                    score = len(matching_symptoms) / len(info["symptoms"])
                    possible_diagnoses.append((diagnosis, score, info["recommendations"]))
            
            possible_diagnoses.sort(key=lambda x: x[1], reverse=True)
            return possible_diagnoses
        ```
    *   **в) Пользовательский интерфейс:**
        ```python
        def main():
            print("=== Экспертная система диагностики ПК ===")
            print("Введите симптомы через запятую (например: медленная работа, всплывающие окна)")
            input_symptoms = [s.strip().lower() for s in input("Симптомы: ").split(',')]
            
            results = diagnose_pc(input_symptoms)
            
            if not results:
                print("\nДиагноз по вашим симптомам не найден.")
                return
            
            print("\n--- Возможные диагнозы ---")
            for diagnosis, score, recommendation in results:
                print(f"\nДиагноз: {diagnosis} (Совпадение: {score:.0%})")
                print(f"Рекомендация: {recommendation}")
        ```
