### Раздел 3 «Алгоритмы и программирование»


#### Вопросы формирующего оценивания усвоения материала


1. **Отличия алгоритмов машинного обучения с полной и минимальной степенью участия человека**  
* **С полной степенью участия (обучение с учителем, supervised learning)**: алгоритм обучается на размеченных данных — для каждого примера есть «правильный ответ».  
  *Пример:* классификация писем на «спам»/«не спам» (есть разметка: какие письма — спам, какие — нет).  
* **С минимальной степенью участия (обучение без учителя, unsupervised learning)**: данных нет разметки; алгоритм сам находит закономерности и структуры.  
  *Пример:* кластеризация пользователей соцсети по поведению (нет заранее заданных групп — алгоритм сам выделяет сегменты).

2. **Определение типа задачи**  
* Предсказание цены на жильё → **регрессия** (прогнозируем числовое значение).  
* Определение сегментов пользователей → **кластеризация** (группируем похожие объекты без заданных классов).  
* Прогнозирование, какой продукт купит клиент → **классификация** (предсказываем категорию из конечного набора).  
* Сортировка писем на «спам»/«не спам» → **классификация** (два класса).

3. **Для какой цели применяются алгоритмы кластеризации?**  
* **Разделение клиентов на группы** (верно).  
* Определение стоимости товара → регрессия.  
* Предсказание температуры → регрессия.

4. **Вставьте пропущенные термины**  
* Машинное обучение на размеченных данных → **обучение с учителем (supervised learning)**.  
* Машинное обучение без разметки → **обучение без учителя (unsupervised learning)**.  
* Сочетание размеченных и неразмеченных данных → **полууправляемое обучение (semi-supervised learning)**.

5. **Метасимвол для любого одиночного символа**  
* **.** (точка).

6. **Функция Python для длины строки**  
* **len()**.

7. **Квантификатор для одного или более вхождений**  
* **+**.

8. **Метасимвол для конца строки**  
* **$**.

### Задания для текущего оценивания усвоения материала

1. **Определите, верно ли утверждение**  

| + или – | Высказывание |
|---------|-------------|
| **–** | Алгоритм классификации используется для предсказания числовых значений |
| **+** | Алгоритм кластеризации группирует данные на основе их сходства |
| **–** | Алгоритмы регрессии могут использоваться для определения категорий объектов |

2. **Соотнесите регулярные выражения с назначением**  

| a | b | c | d |
|---|---|---|---|
| 1 | 2 | — | 3 |

* **a) \d+** → 1. Одно или более цифровых символов.  
* **b) \w{2,5}** → 2. Любое слово длиной от 2 до 5 символов.  
* **c) ^a.*z$** → Строка начинается с «a» и заканчивается на «z» (в таблице нет номера, оставляем как есть).  
* **d) [A-Za-z]{3,}** → 3. Слово из букв латинского алфавита длиной не менее 3 символов.

3. **Регулярное выражение для телефонных номеров «8‑XXX‑XXX‑XX‑XX»**  
```regex
8-\d{3}-\d{3}-\d{2}-\d{2}
```
* *Объяснение:*  
  * `8-` — буквальное совпадение.  
  * `\d{3}` — три цифры.  
  * `-` — дефис.  
  * Повторяем шаблон для остальных групп.

4. **Код для удаления всех цифр из строки**  
```python
import re

text = "Текст с 12345 цифрами"
result = re.sub(r'\d+', '', text)
print(result)  # "Текст с  цифры" (между "с" и "цифры" — пробел от удалённых цифр)
```
* *Примечание:* Если нужно убрать лишние пробелы, можно дополнительно применить `re.sub(r'\s+', ' ', result).strip()`.

5. **Регулярное выражение для слов, начинающихся с «А»**  
```regex
\bA[а-яА-ЯёЁ]*\b
```
* *Объяснение:*  
  * `\b` — граница слова (чтобы не захватывать части слов).  
  * `A` — буква «А» (латинская или кириллица — уточните, если нужно только кириллицу).  
  * `[а-яА-ЯёЁ]*` — любые буквы кириллицы после «А».  
  * `\b` — конец слова.  
* Для примера «Апельсин, банан, Ананас» найдёт: «Апельсин», «Ананас».


6. **Регулярное выражение для дат «дд.мм.гггг»**  
```regex
\d{2}\.\d{2}\.\d{4}
```
* *Объяснение:*  
  * `\d{2}` — две цифры (день).  
  * `\.` — точка (экранирована).  
  * `\d{2}` — две цифры (месяц).  
  * `\.` — точка.  
  * `\d{4}` — четыре цифры (год).  
* Для текста «05.07.2024, 15.08.2024» найдёт обе даты.
